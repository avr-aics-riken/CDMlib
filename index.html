Welcome to CDMlib

The Cartesian IData Management Library is a C++ class library that manages file I/O for Cartesian grid data, which is a former CIOlib. New features are incorporated in CDMlib such as plot3d file format, and non-uniform Cartesian data.

Functions

CDMlib offers the following functions:

  • Management of grid and domain decomposition information using DFI metadata
  • SPH, BOV, and Plot3d function file format support
  • MxN loading (for different numbers of parallel executions)
  • Data loading from coarse to fine meshes (for cases in which the fraction of grids in each direction is 1/2 – or 1/8 for 3D)
  • Staging (copying files to a directory by rank using an external program)
  • External distributed parallel file conversion support (SPH, BOV, PLOT3D → SPH, BOV, PLOT3D, AVS, VTK)
